Executive Summary — ISO 27001 Risk Assessment

This assessment looks at the organisation’s current security risks using the ISO/IEC 27001:2022 framework. The goal is to highlight the main issues and outline simple, practical steps to reduce risk.

The review found several gaps: weak password practices, no MFA, no centralised logging or monitoring, inconsistent patching, limited vendor reviews, and no formal incident response plan. These issues make it harder for the organisation to prevent, detect, and respond to security incidents.

Overall, the risk level is Moderate to High, mainly due to weaknesses in access control, monitoring, and general governance.

Recommended improvements include enabling MFA, setting up centralised logging, creating a proper patching process, reviewing key vendors regularly, and building and testing an incident response plan. These steps will help strengthen the organisation’s security posture moving forward.
